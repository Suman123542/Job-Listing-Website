# JobSeekerr

JobSeekerr is a simple job listing website where users can search for jobs, view job details, and apply for positions.  
The website is fully responsive and works smoothly on mobile and desktop devices.

## What I Did

- Created a job listing website using React
- Added job search by keyword and location
- Implemented a floating job details section
- Added apply functionality with success alert
- Built responsive navigation with mobile menu
- Created Companies and Blog pages
- Designed clean and modern UI

## What I Used

- React
- React Router
- Tailwind CSS
- JavaScript
